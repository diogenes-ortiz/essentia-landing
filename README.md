# Essentia Landing Prototype

Static prototype for the Mi Pileta Essentia landing page.

## Local preview

```powershell
node work/serve.mjs
```

Then open:

```text
http://localhost:4173/
```

## GitHub Pages

This repository includes a GitHub Actions workflow that publishes the `outputs/` folder to GitHub Pages on every push to `main`.

After pushing the repository to GitHub, enable Pages with **Source: GitHub Actions** in the repository settings.
